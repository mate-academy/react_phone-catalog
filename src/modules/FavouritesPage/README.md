# FavouritesPage

Сторінка для відображення обраних (улюблених) товарів користувача.

## Функціональність

- **Відображення списку обраних товарів** через компонент `ProductCard`
- **Кнопка Back** - повертає на попередню сторінку (використовує Browser Back)
- **Лічильник товарів** - показує кількість обраних товарів
- **Пустий стан** - відображається, коли немає обраних товарів:
  - Іконка сердечка
  - Повідомлення "Your favourites list is empty"
  - Підказка "Add products you like to see them here"

## Використання

Доступна за шляхом: `/#/favorites`

## Адаптивність

- **Mobile**: 1 колонка
- **Tablet**: 2 колонки
- **Desktop**: 4 колонки

## Компоненти

- `ProductCard` - відображення товару з можливістю додати в кошик/видалити з обраного
- `Button` - кнопка Back
- `Icon` - іконки для UI

## Контекст

Використовує `useFavorites()` hook для:

- Отримання списку обраних товарів `favorites`
- Робота з обраними (додавання/видалення) відбувається в `ProductCard`
