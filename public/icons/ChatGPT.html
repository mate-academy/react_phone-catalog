Run 'npm i -S react-redux' to add iteslintimport/no-extraneous-dependencies

Помилка, яку ви бачите, вказує на те, що бібліотека react-redux не включена у залежності вашого проєкту. Для виправлення цієї помилки вам потрібно додати react-redux до залежностей у вашому файлі package.json.

Виконайте команду в терміналі, щоб встановити react-redux:

sh
npm install --save react-redux

Або скористайтесь скороченою командою:

sh
npm i -S react-redux

Після цього ESLint перестане показувати цю помилку, оскільки бібліотека буде правильно додана до залежностей вашого проєкту.

Переконайтесь, що у вас у файлі package.json після встановлення з'явиться відповідний запис у розділі dependencies:

json
"dependencies": {
  "react": "^17.0.2",
  "react-dom": "^17.0.2",
  "react-redux": "^7.2.4",  // версія може бути інша, в залежності від останньої доступної
  "react-router-dom": "^6.2.1",
  "redux": "^4.1.0"
}

Після виконання вказаних дій проблема повинна зникнути.
