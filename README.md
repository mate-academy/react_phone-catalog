# React Product Catalog

This project showcases a modern and responsive product catalog application built with React. The website provides smooth navigation, detailed product pages, shopping cart functionality, and a favorites system for a complete e-commerce experience.

Users can browse phones, tablets, and accessories, explore product details, add products to favorites or cart, and enjoy dynamic sorting, filtering, and pagination features.

Experience the live website: [React Product Catalog](https://u5135039754-dev.github.io/react_phone-catalog/)

# Technologies Used

Core

- React {v18} — Frontend library
- TypeScript — Static typing
- SCSS Modules — Component-based styling
- JavaScript (ES6+) — Interactivity and logic
- React Router — Navigation and routing
- Context API / Redux — State management
- LocalStorage — Saving cart and favorites data

# Features

- Responsive and modern UI
- Product categories: Phones, Tablets, Accessories
- Dynamic product sliders
- Product details pages with image gallery
- Shopping cart with quantity management
- Favorites page
- Sorting, filtering, and pagination
- Product search with debounce
- Smooth hover animations and transitions
- Breadcrumb navigation
- Error handling and loading states
- Optional dark/light theme switching

# Project Structure

- `src/components` — Reusable UI components
- `src/modules` — Page-based modules
- `src/pages` — Main application pages
- `src/context` / `src/store` — Global state management
- `public/api` — Product data
- `public/img` — Product images

# Pages Included

- Home Page
- Product Category Pages
- Product Details Page
- Shopping Cart Page
- Favorites Page
- Not Found Page

# Additional Functionality

- Cart and favorites saved in `localStorage`
- Smooth scrolling and transitions
- Automatic image slider
- Product suggestions section
- Fully responsive design
