# Nice Gadgets Store

# [Demo](https://gadgetstore-git-develop-heorhiis-projects-1d896517.vercel.app/)

Portfolio project: an online catalog concept for browsing phones, tablets, and accessories.

## Project Overview

This project simulates a modern gadget store experience with a clean interface, smooth navigation, and mobile-friendly layout.

It was created to showcase product thinking and frontend development skills through a realistic e-commerce scenario.

## What Visitors Can Do

- Explore a homepage with promo banners and featured products
- Browse a working phone catalog with product cards
- Navigate by category (phones, tablets, accessories)
- Switch between light and dark themes
- Use the site on desktop and mobile screens

## Current Status

Implemented:
- Full app layout and navigation
- Home page sections and promotional slider
- Phones catalog page with product listing
- Responsive header, mobile menu, and footer
- Theme preference saving (light/dark)

In progress:
- Full tablets and accessories catalog experience
- Favorites and shopping bag pages
- Product details pages
- Advanced browsing tools (search, filters, sorting, pagination)

## Why This Project Matters

- Demonstrates how to turn a design idea into a real, interactive product
- Focuses on usability, responsiveness, and visual consistency
- Uses reusable component architecture to support future scaling
