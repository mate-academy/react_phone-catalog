# React Phone Catalog

A **modern, responsive React project** simulating a gadget store.
This project demonstrates skills in **React development, SCSS styling, responsive design, and frontend architecture**.
All data is currently static, but the project is structured for **easy backend integration** and feature expansion.

---

##  Key Features

- **Category Pages:** Phones, Tablets, Accessories
- **Favorites Page:** Mark products as favorites
- **Cart Page:** Add/remove items
- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile
- **Fast Navigation:** React Router for smooth routing
- **Reusable Components:** Buttons, Cards, Layouts, Modals
- **State Management:** Local component state and prop drilling
- **Accessible & Semantic HTML**

---

##  Tech Stack & Skills Demonstrated

**Frontend & Styling:**
- React (Functional Components + Hooks)
- React Router
- TypeScript (ES6+)
- SCSS / Sass
- CSS Modules & BEM methodology
- Responsive design (Flexbox, Grid, Media Queries)
- Web Fonts & Asset Management

**Development Skills:**
- Git & GitHub workflows (branching, PRs, merging)
- Code organization & modular architecture
- Prettier & ESLint for code quality
- StyleLint for CSS/SCSS conventions
- Component reusability & DRY principles

**Optional/Advanced:**
- Ready for **backend integration** (API calls, fetch/axios)
- Supports adding checkout flows
- Can be extended with **Redux / Context API** if needed
