# React Product Catalog

A modern React-based product catalog application that allows users to browse phones, tablets, and accessories, view detailed product information, manage favorites, and add products to a shopping cart.

## Live Preview

🔗 https://maksymivivan2000-dotcom.github.io/react_phone-catalog/

## Design Reference

🔗 https://www.figma.com/file/T5ttF21UnT6RRmCQQaZc6L/Phone-catalog-(V2)-Original

## Technologies Used

* React
* TypeScript
* React Router
* SCSS Modules
* Context API
* LocalStorage
* HTML5
* Git & GitHub

## Getting Started

### Clone the repository

```bash
git clone https://github.com/maksymivivan2000-dotcom/react_phone-catalog.git
cd react_phone-catalog
```

### Install dependencies

```bash
npm install
```

### Run the project locally

```bash
npm start
```

## Features

* Product catalog with phones, tablets, and accessories
* Product details pages
* Favorites management
* Shopping cart functionality
* Product search
* Product sorting
* Pagination
* Breadcrumb navigation
* Responsive design
* Data persistence using LocalStorage
* Not Found pages
* Suggested products section
* Smooth hover animations
