**Phone Catalog** is a responsive e-commerce web application for electronics, built with React's component-based approach and robust state management powered by Redux. The interface features a clean, minimalist design with a focus on seamless UI/UX to ensure a smooth shopping experience.

##  Main Features

* **Product Catalog:** Seamlessly browse through available tech gadgets with detailed specifications, models, and up-to-date pricing.
* **Cart System:** Add items to the shopping cart with a dynamic live recalculation of total prices and item counts.
* **Favorites / Wishlist:** Save items you like to a dedicated favorites panel for quick access and later consideration.
* **Filtering & Sorting:** Efficiently narrow down choices by categories, brands, prices, or key specifications.
* **Fully Responsive:** Beautifully optimized layout ensuring an ideal user experience across mobile phones, tablets, and desktop displays.

## Technology Stack
* React, Redux Toolkit, SCSS
